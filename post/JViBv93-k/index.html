C++实现三维向量及其计算

2019-08-01

数据的爆炸增长，二维的状态的向量已经无法满足日常的需求，三维图像慢慢的步入研究领域，本文使用C++实现了一个三维向量的定义和部分的运算实现。

引言

三维图像是目前在机器视觉和人工智能领域中应用比较广泛的一个数学基础，从近几年的国内外会议论坛文章可见一斑，笔者的一个项目里面包含三维的计算，就创建了一个三维向量类，但是并未完全实现所有的向量运算，如有需要者可以按照自己的需求增加。

方法论

向量是我们在高中学习的一个重要的数学概念，点是终点描述的是位置（相对位置），向量描述的是大小和方向，也可以描述相对位置，所以向量可以代替点，点不能代替向量，且有些计算不需要知道原点，用向量来描述3D几何计算更加灵活，3D中的所有于位置、大小、方向、方位等几何数学相关的计算都应该优先考虑用向量来表达[1]^{[1]}[1]。关于点和向量的关系，可以由下图表示

在研究向量之前需要补充一下向量计算的基础。

向量的坐标表示[2]^{[2]}[2]

使用两个点的坐标就可以计算向量，假设有两点P1和P2，以P1作为基准点，向量V=P2-P1，假如以P2作为基准点，则向量V=P1-P2，总之V=P-Pb，Pb为基准点。

V=P−Pb=(x−xb,y−yb,z−zb)=(a,b,c){V=P-Pb=(x-xb,y-yb,z-zb)=(a,b,c)} V=P−Pb=(x−xb,y−yb,z−zb)=(a,b,c)

举例：假设P1=（1，2，3），P2=（2，2，2），将P2看作基准点，那么根据公式：

V=P1−P2=(1−2,2−2,3−2)=(−1,0,1){ V=P1-P2=(1-2,2-2,3-2)=(-1,0,1) } V=P1−P2=(1−2,2−2,3−2)=(−1,0,1)

可以看出，虽然P1和P2的三个分量都处于正轴，但是由于基准点取的是P2，所以V的向量在x方向上为指向x轴的负轴，而V的b值是0，因此向量处于一个XZ平面上，如图：

将V放入坐标中

向量的运算

笔者主要用到的几个运算分别是：加、减、叉乘、点乘五个运算。

加法和减法运算

加法和减法运算是最简单的云算法则，直接按位加减即可，例如：

a⃗±b⃗=(1,2,3)±(4,5,6)=(1±4,2±5,3±6){\vec{a} \pm \vec{b}} = (1,2,3) \pm (4,5,6) = (1 \pm 4,2 \pm 5,3 \pm 6) a±b=(1,2,3)±(4,5,6)=(1±4,2±5,3±6)

叉乘运算

向量的叉乘云算是一个特殊的计算方式，叉乘结果仍是向量[3]^{[3]}[3]，公式如下：

a⃗×b⃗=(a1,a2,a3)×(b1,b2,b3)=(a2b3−a3b2,a3b1−a1b3,a1b2−a2b1){\vec{a} \times \vec{b} = (a{_1},a{_2},a{_3}) \times (b{_1},b{_2},b{_3})=(a{_2}b{_3} - a{_3}b{_2},a{_3}b{_1} - a{_1}b{_3},a{_1}b{_2} - a{_2}b{_1})} a×b=(a1​,a2​,a3​)×(b1​,b2​,b3​)=(a2​b3​−a3​b2​,a3​b1​−a1​b3​,a1​b2​−a2​b1​)

实例：

a⃗×b⃗=(1,2,3)×(4,5,6)=(2×6−3×5,3×4−1×6,1×5−2×4)=(−3,6,−3){\vec{a} \times \vec{b} = (1,2,3) \times (4,5,6) = (2 \times 6 -3 \times 5,3 \times 4 -1 \times 6,1 \times 5 -2 \times 4) = (-3,6,-3)} a×b=(1,2,3)×(4,5,6)=(2×6−3×5,3×4−1×6,1×5−2×4)=(−3,6,−3)

点乘运算

点乘运算时最常用的一个运算，用于计算向量夹角、线面夹角、绵绵夹角等各种情况，计算规则比较简单，计算结果是一个数字。通常用于表示a⃗{\vec{a}}a向量在b⃗{\vec{b}}b向量方向上的投影长度（同方向为正反方向为负号）[4]^{[4]}[4]，公式如下：

a⃗⋅b⃗=∑i=1naibi=a1b1+a2b2+a3b3+...anbn{\vec{a} \cdot \vec{b} = \sum_{i=1}^na_ib_i=a_1b_1+a_2b_2+a_3b_3+...a_nb_n} a⋅b=i=1∑n​ai​bi​=a1​b1​+a2​b2​+a3​b3​+...an​bn​

实例：

a⃗⋅b⃗=(1,2,3)⋅(4,5,6)=1×4+2×5+3×18=32{\vec{a} \cdot \vec{b} = (1,2,3) \cdot (4,5,6) = 1 \times 4 + 2 \times 5 + 3 \times 18 = 32} a⋅b=(1,2,3)⋅(4,5,6)=1×4+2×5+3×18=32

向量的夹角运算

向量夹角运算是采用的点乘的几何定义，在欧几里得空间中，点积可以直观地定义为[5]^{[5]}[5]：

a⃗⋅b⃗=∣a⃗∣∣b⃗∣cosθ{\vec{a} \cdot \vec{b} = |\vec{a}||\vec{b}|cos\theta} a⋅b=∣a∣∣b∣cosθ

这里 |x⃗{\vec{x}}x| 表示x⃗{\vec{x}}x的模（长度），θ{\theta}θ 表示两个向量之间的角度、

此时可以根据公式推导出：

θ=arccosa⃗⋅b⃗∣a⃗∣∣b⃗∣{\theta = arccos \frac{\vec{a} \cdot \vec{b}}{|\vec{a}||\vec{b}|}} θ=arccos∣a∣∣b∣a⋅b​

注意不同程序语言计算的θ\thetaθ不同，需要注意角度和弧度问题。

a⃗//b⃗<=>a⃗=kb⃗<=>a1b1=a2b2=a3b3=k{\vec{a} // \vec{b} <=> \vec{a} = k\vec{b} <=> \frac{a_1}{b_1} = \frac{a_2}{b_2} = \frac{a_3}{b_3} = k} a//b<=>a=kb<=>b1​a1​​=b2​a2​​=b3​a3​​=k

a⃗⋅b⃗<=>a1b1+a2b2+a3b3=0{\vec{a} \cdot \vec{b} <=> a_1b_1 + a_2b_2 + a_3b_3 = 0} a⋅b<=>a1​b1​+a2​b2​+a3​b3​=0

实现

前面啰嗦这么多完全是为了让你能够看懂代码，程序员估计已经安奈不住自己的迫切需求，下面给出实现，实现程序采用C++ 17,CLion项目，每一个文件的代码如下，也可以下载附件源码：

//Vector3D.h
#ifndef DBM_VECTOR3D_H
#define DBM_VECTOR3D_H

class Vector3D {
public:
    signed int X,Y,Z;
    Vector3D();
    Vector3D(int i);
    Vector3D(int x,int y,int z);
    ~Vector3D();
    void print();
    Vector3D operator*(double lamda);
    double operator*(const Vector3D &v3);
    Vector3D operator+(const Vector3D &v3);
    Vector3D operator-(const Vector3D &v3);
    double getModule();
    double getDotMulti(Vector3D v3);
    Vector3D getXMulti(Vector3D v3);
    double getDelta(Vector3D v3,bool isRadian = true);
    double getAbsoluteDelta(Vector3D v3,bool isRadian = true);
    double isCollinear(Vector3D v3);
    double getLength();
};
#endif //DBM_VECTOR3D_H

//Vector3D.cpp
#include "Vector3D.h"

Vector3D::Vector3D() {}

Vector3D::Vector3D(int i) { this->X = i, this->Y = i, this->Z = i; }

Vector3D::Vector3D(int x, int y, int z) {
    this->X = x;
    this->Y = y;
    this->Z = z;
}

Vector3D Vector3D::operator+(const Vector3D &v3) { return Vector3D(this->X + v3.X, this->Y + v3.Y, this->Z + v3.Z); }

Vector3D Vector3D::operator-(const Vector3D &v3) { return Vector3D(this->X - v3.X, this->Y - v3.Y, this->Z - v3.Z); }

double Vector3D::operator*(const Vector3D &v3) { return (this->X * v3.X + this->Y * v3.Y + this->Z * v3.Z); }

Vector3D Vector3D::operator*(const double lamda) { return Vector3D(this->X * lamda, this->Y * lamda, this->Z * lamda); }

double Vector3D::getModule() { return sqrt(pow(this->X, 2) + pow(this->Y, 2) + pow(this->Z, 2)); }

double Vector3D::getLength() { return this->getModule(); }

double Vector3D::getDotMulti(Vector3D v3) { return (this->X * v3.X + this->Y * v3.Y + this->Z * v3.Z); }

Vector3D Vector3D::getXMulti(Vector3D v3) {
    //vectorA x vectorB = (AyBz - AzBy, AzBx - AxBz, AyBz - AzBy)。
    return Vector3D(this->Y * v3.Z - this->Z * v3.Y,
                    this->Z * v3.X - this->X * v3.Z,
                    this->Y * v3.Z - this->Z * v3.Y);
}

double Vector3D::getDelta(Vector3D v3, bool isRadian) {
    double c = (getDotMulti(v3)) / (getModule() * v3.getModule());
    double iResult = acos(c);
    if (!isRadian)
        iResult = iResult * 180 / 3.1415926535898;
    return iResult;
}

double Vector3D::getAbsoluteDelta(Vector3D v3, bool isRadian) { return abs(getDelta(v3, isRadian)); }

void Vector3D::print() { printf("(%d,%d,%d):%lf\n", this->X, this->Y, this->Z, getModule()); }

double Vector3D::isCollinear(Vector3D v3) {
    return false;
}

Vector3D::~Vector3D() {}
