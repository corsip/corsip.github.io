Java中的SSM框架配置
2018-09-23

SSM框架是JavaWeb中常用的框架，包括Spring、Spring MVC、Mybatis三种框架的一个整合。其中任意两个框架都可以相互组合SS框架、SM（O(∩_∩)O~）框架（两种情况：Spring + Mybatis、Spring MVC + Mybatis）。具体的详细情况可以参考这篇文章http://blog.csdn.net/daniel_fei/article/details/56670751。
